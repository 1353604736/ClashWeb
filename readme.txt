1. 便携版不需要安装pytHon。

2. 然后就可以启动了！！ 
    双击ClashWeb.bat 自动打开浏览器以及python后台
    当不需要开/关 Clash 更新配置文件等操作时,可以通过【输入2】关闭python后台。
